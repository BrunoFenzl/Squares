    mq-up

    @mixin mq-up($name, $bps) { ... }

    Description

    Media of at least the minimum breakpoint width. No query for the smallest breakpoint. Makes the @content apply to the given breakpoint and wider. Taken from Bootstrap 4

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    Name of the breakpoint. should be defined the $grid-breakpoints map.

    		String— none
    $bps

    Map of Breakpoint names and widths. Defaults to $grid-breakpoints.

    		Map— none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Output

    Content of the mixin.

    Requires

    • [function] mq-min

    Used by

    • [mixin] make-props

    mq-min

    @function mq-min($name, $bps) { ... }

    Description

    Find the width of given breakpoint key and return it or null if width == 0. Taken from Bootstrap 4

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    Name of the breakpoint. should be defined the $grid-breakpoints map.

    		String— none
    $bps

    Map of Breakpoint names and widths. Defaults to $grid-breakpoints.

    		Map— none

    Returns

    Number or or Null —

    Minimum breakpoint width. Null for the smallest (first) breakpoint.
